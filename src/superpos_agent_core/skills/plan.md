---
name: plan
description: Create an implementation plan for a feature or task
---

Create a step-by-step implementation plan for the given task. Include:

1. **Goal** — what we're building in one sentence
2. **Steps** — numbered list of concrete implementation steps
3. **Files to create/modify** — list affected files
4. **Edge cases** — things to watch out for
5. **Testing** — how to verify it works

Keep it practical and actionable. No boilerplate — just the plan.
